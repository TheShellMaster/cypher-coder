# 🤖 Cypher AI — Assistant Web Autonome

**Cypher AI** est l'assistant de programmation web autonome de la famille Cypher, conçu pour répondre à vos questions complexes, concevoir des algorithmes de haut niveau et effectuer des recherches sur internet en temps réel.

Développé par **DJAKOUA KWANKAM**, étudiant en informatique à l'**Institut Universitaire de Technologie de Douala (IUT)**.

## 🚀 Fonctionnalités
*   🤖 **Modèle d'IA** : Alimenté par `Qwen/Qwen2.5-Coder-7B-Instruct` sur le serveur libre de Hugging Face.
*   🌐 **Recherche Web Intégrée** : Recherche automatique via DuckDuckGo lorsque requis par la conversation.
*   📊 **Télémétrie & Apprentissage** : Sauvegarde confidentielle des interactions pour alimenter le dataset privé d'entraînement.
